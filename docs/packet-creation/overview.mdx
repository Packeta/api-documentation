# Overview

# Packet creation

Packets can be created in multiple ways:
- By using our [client section](https://client.packeta.com) (either manually
using [this form](https://client.packeta.com/packet-drafts/), or using a [`CSV`](../getting-started/client-section-imports#csv-import)/
[`XML`](../getting-started/client-section-imports#xml-import) data upload)

- Using our [`REST/XML`](../getting-started/packetery-api#restxml)/
[`SOAP`](../getting-started/packetery-api#soap) API, specifically using the 
[`createPacket()`](../api-reference/api-methods-reference#createpacket) method. This method
should be used for all parcel creation, regardless of the delivery service you wish to use. 

When creating parcels, you can choose the delivery service based on which [`PacketAttributes`](../api-reference/data-structures-reference#packetattributes) you pass when creating
a parcel.
