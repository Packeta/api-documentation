# Carriers' validation

Some external carriers do have specific limitations and properties which we aim to document in this section.

## Common rules
- weight is **mandatory** for all carriers (value must be bigger than 0)
- allowed label print formats:
  - A6 on A4
  - A6 on A6

## Possible errors
- List of known error messages with details can be found at [Packeta API Reference](../api-reference/errors-reference)

Known restrictions are listed for relevant carriers.

## Carrier validations

### AT Austrian Post HD
- weight validation (max. 30kg)

### BG Sameday Box
- require filled in packet dimensions

### BG Speedy HD
- recipient name validation (min. 3 characters)
- address validation (zip code, city)

### BG Speedy PP
- name validation (min. 3 characters)
- address validation (zip code, city)

### BE Dutch Post HD
- weight validation (max. 20kg)

### CZ Packeta HD

### DE Green Delivery

### ES Correos HD
